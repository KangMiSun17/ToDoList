# 주제

TypeScript를 이용해서 TodoList 만들기

# 개요

TypeScript를 공부하는데 이론만 공부하니 감이 잡히지 않아서 뭔가를 만들어보자! 하고 저번에 만들다 localStorage 연결에 실패한 TodoList를 도전해보았다.

# 기능

- 오늘 날짜와 요일 보여주고 남은 할 일 개수 확인
- 완료하면 체크 표시와 함께 글자 색 변화
- 할 일 추가와 삭제
